Installation

From Source

The source lives at https://github.com/csound-plugins/csound-plugins

Dependencies

  • a compiler
  • cmake
  • csound >= 6.13
git clone https://github.com/csound-plugins/csound-plugins
cd csound-plugins
mkdir build
cd build
cmake ..
make 
sudo make install
